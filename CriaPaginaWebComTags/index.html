Listas

Você pode criar listas ordenadas e não ordenadas usando as tags <ol> e <ul> respctivamente, juntamento com <li> para os itens da lista.

Links

Use a tag <a> para criar hyperlinks. O atributo href define o destino do link.

Visite o Site de Exemplo para mais informação.

A tag <hr> é usada para inserir uma linha horizontal que pode ser usada para separar conteúdo.
